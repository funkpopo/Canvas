# Dashboard Data Points

## Cluster Overview
- Kubernetes version
- Cluster name/ID
- API server availability

## Node Metrics
- Node count and status
- CPU / memory allocatable vs usage
- Labels and taints summaries

## Namespace Metrics
- Namespace inventory
- Resource quotas and usage
- Recent events

## Workloads
- Deployments, StatefulSets, DaemonSets, Jobs
- Replica status, rollout history, restart counts

## Health & Alerts
- Pod health summary
- Pending / failed Pods
- Cluster events stream

## Networking
- Services and ingress resources
- Service endpoint health

## Storage
- PersistentVolume claims and status

## Future Enhancements
- Policy compliance scores
- Cost and capacity projections
